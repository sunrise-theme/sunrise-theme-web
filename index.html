Install Source

A beautifully fresh dual theme for Visual Studio Code

Installation

  1. Open the extensions sidebar in Visual Studio Code.
  2. Search for "Sunrise Theme".
  3. Click install.
  4. Open the Command Palette with Ctrl+Shift+P or ⇧⌘P .
  5. Select "Preferences" -> "Color Theme" , and choose a "Sunrise" variant.
  6. Enjoy!
